Kiefer-Wolfowitz#

The Kiefer-Wolfowitz algorithm is a stochastic algorithm for finding an extremum of an unknown function \(f\) from \(\mathbb R^d\) to \(\mathbb R\).

The algorithm do the following assumptions

  • \(f\) is strictly concave,

  • \(f\) is differentiable,

and is defined by

\[X_{n+1} = X_n + \frac{\gamma_n}{2c_n}(Y_{n+1} - Z_{n+1})\]

where the initial state \(X_0 \in \mathbb R^d\) is chosen arbitrarily, and \((\gamma_n)\), \((c_n)\) are two deterministic, positive, and decreasing sequences converging to zero, such that

\[\sum_{n = 1}^{+∞}\gamma_n = +∞, \qquad \sum_{n = 1}^{+∞}\gamma_n c_n < +∞, \qquad \sum_{n = 1}^{+∞} \left (\frac{\gamma_n}{c_n} \right)^2 < +∞.\]

The implementation of this algorithm in Algosto is such that \(\gamma_n = \frac{a}{n^\alpha}\) and \(c_n = \frac{b}{n^\beta}\).

The sequences \((Y_n)\) and \((Z_n)\) are two random vector sequences of \(\mathbb R^d\) satisfying

\[\begin{split}\mathbb E[Y_{n+1}|F_n] = \begin{pmatrix} f(X_n + c_ne_1) \\ f(X_n + c_ne_2) \\ \vdots \\ f(X_n + c_ne_d) \end{pmatrix}, \qquad \mathbb E[Z_{n+1}|F_n] = \begin{pmatrix} f(X_n - c_ne_1) \\ f(X_n - c_ne_2) \\ \vdots \\ f(X_n - c_ne_d) \end{pmatrix}\end{split}\]

where \((e_1, e_2, \dots, e_d)\) is the canonical basis of \(\mathbb R^d\).

The Kiefer-Wolfowitz algorithm wrapped in a solver class.

The objective function to minimize. This function must be able to handle a 1-D array of several points.

The a value as described below.

The alpha value as described below.

bfloat

The b value as described below.

The beta value as described below.

References

[1]

Kiefer, J. and Wolfowitz, J., “Stochastic estimation of the maximum of a regression function”, Annals of Mathematical Statistics, 1952.

...    # ...

>>> solver.fit()
